❓ Aide

Comment utiliser Champipi efficacement

📸 Identification

Il existe plusieurs façons d’identifier un champignon avec Champipi :

🔹 Recherche par critères

Sélectionne un critère (par exemple Chapeau), puis entre un ou plusieurs termes (ex. : rouge, écailleux).

Tu peux combiner plusieurs critères en ajoutant d’autres filtres (ex. : Chapeau = rouge, Pied = blanc).

🔹 Recherche par description libre

Tu peux aussi écrire directement une description comme : chapeau rouge, pied blanc, face poroïde bleue.

L’application analysera automatiquement ta phrase et créera les bons filtres.

🧠 Astuces pour affiner ta recherche

  • Virgule = ET → tous les termes doivent être présents.
    Exemple : chapeau rouge, écailleux (les deux conditions doivent être vraies)
  • OU → au moins un des termes doit être présent.
    Exemple : chapeau rouge OU jaune
  • Exclusion avec ! → exclut un terme.
    Exemple : chapeau rouge OU jaune, !vert = rouge ou jaune, mais pas vert

🖼️ Suggestion par photo (IA)

Tu peux aussi prendre une photo : l’intelligence artificielle te proposera des mots-clés pertinents à utiliser pour ta recherche.

⭐ Favoris

Ajoute une espèce en favori pour y accéder rapidement plus tard.

🗺️ Observations

Consulte une carte avec la position de tes trouvailles.

📶 Mode hors-ligne

Les principales fonctionnalités sont disponibles même sans connexion.
